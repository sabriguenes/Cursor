## Guides

### Enterprise Privacy & Security

The #1 question enterprises ask: *"Will our proprietary code be safe?"*

| Guide | Language | Description |
|-------|----------|-------------|
| [Enterprise Privacy Guide](enterprise/privacy-guide-en.md) | English | Deep-dive into Cursor's privacy architecture, Merkle trees, dual infrastructure |
| [Enterprise Datenschutz-Leitfaden](enterprise/privacy-guide-de.md) | Deutsch | Deutsche Version des Privacy Guides |

**Key topics covered:**
- Privacy Mode vs Share Data
- What's actually stored (embeddings, file paths, hashes)
- Zero Data Retention agreements with OpenAI, Anthropic, etc.
- GDPR compliance and EU SCCs
- Practical `.cursorignore` recommendations
- Network controls and domain whitelisting
